# Who-Wants-to-Be-a-Millionaire
This is a Java GUI of the game 'Who Wants to Be a Millionaire?' built by Johnny Dong and Ethan Haggart for the Program Design and Construction Paper.

#
![image](https://user-images.githubusercontent.com/87466526/174991622-b445ae48-52f9-4a5e-8cb5-cf859bf9db1b.png)

## Functionalities
### Audience Lifeline:
An accurate option will be highlighted in red to assist the player with selecing the answer, once the 'Audience Lifeline' button is pressed. This funcitonality can only be used once per game.

![image](https://user-images.githubusercontent.com/87466526/174991976-ac2152eb-ea2f-40ca-8163-c25685c79446.png)

### 50-50 Lifeline:
Two of the incorrect options will be disabled. The player will be left with only two options, where one option is incorrect and the other is the answer. This funcitonality can only be used once per game.

![image](https://user-images.githubusercontent.com/87466526/174993630-037e616f-d21b-4796-9bcd-e4093ce8979d.png)

### Control Prompt:
The controls of the game will be prompted on screen when the 'Controls' button is pressed.

![image](https://user-images.githubusercontent.com/87466526/174994014-0b9d9fd9-de81-43f0-8d4e-09beda8d1dd1.png)
